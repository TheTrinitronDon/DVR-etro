# DVR-etro
Aiming to repurpose consumer-grade DVR systems with Embedded Linux SoCs. These systems have tremendous potential to be a flexible retro gaming platform with generous SDRAM, H.264 CODEC, Analog RGB/VGA/S-Video/CBVS processing, SATA hard drive support, USB Mouse &amp; Keyboard drivers, LAN/WLAN capability, and easily found for $10-20USD at thrift stores.
